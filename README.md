# 🏥 Hospital Dashboard | Power BI

## Overview
An interactive Power BI dashboard for hospital management, 
providing insights into patient flow, department performance, 
and satisfaction scores.

## Key Metrics
- Total Patients: 660
- Average Wait Time: 35.98 min
- Average Patient Satisfaction: 1
- Total Departments: 8
- Total Case Management: 20

## Features
- Patient distribution by department
- Gender analysis (Male / Female / NC)
- Monthly patient count tracking
- Patient satisfaction score by department
- Detailed patient table with filters

## Departments Covered
General Practice, Orthopedics, Cardiology, 
Physiotherapy, Neurology, Gastroenterology, Renal

## Filters
Full Name | Patient Age | Department Referral | Month

## Tools Used
- Power BI Desktop
- Excel / SQL Server (Data Source)
